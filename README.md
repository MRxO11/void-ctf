VOID-CTF

A free web based Capture The Flag platform for practicing real world web security vulnerabilities.

Live:  https://void-ctf.vercel.app 

---

## About the Project

VOID CTF is designed to provide hands on experience with common web security flaws.  
The platform contains intentionally vulnerable labs that simulate real application mistakes.

Current lab categories include:

- IDOR (Insecure Direct Object Reference)
- SQL Injection
- Cross-Site Scripting (XSS)
- Logic flaws
- Race conditions
- Authentication and authorization issues

This project is intended for:

- Security learners
- CTF players
- Students
- Pentesting practice
- Security research
